
# Logs Wine Project 🍷

A custom-built e-commerce platform designed for wine retail, optimized for the client's current business model.

### 🚀 Project Scope & Features
This project was developed as a high-performance MVP (Minimum Viable Product) to facilitate sales while maintaining low operational costs for the client.

*   **Product Catalog:** A dynamic display of available wine inventory.
*   **WhatsApp Integration:** A "Direct-to-WhatsApp" ordering system that allows customers to initiate purchases instantly with the vendor.
*   **Email Ordering:** Integrated SMTP functionality to capture order details and send them directly to the store's management.
*   **Scalable Architecture:** Built with a Django backend, allowing for a future transition to a full-scale automated payment gateway (like Paystack or Flutterwave) when the business scales.

### 🛠 Tech Stack
*   **Backend:** Python / Django
*   **Frontend:** HTML5, CSS3, Responsive Design
*   **Deployment:** Managed via Hostinger
*   **Tools:** PyCharm, Git, FileZilla
